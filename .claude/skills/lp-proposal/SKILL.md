# LP提案自動生成 SKILL v2.2

## 最重要ルール
```
1. 各Phaseの完了後、必ずユーザーの確認を待ってから次に進む
2. ユーザーが「OK」「進めて」等の承認を返すまで次のPhaseを開始しない
3. 一度に複数のPhaseを実行しない
4. ユーザーが手直しした内容を勝手に元に戻さない
```

---

## 起動時：スタートメニューを表示する

このSKILLが呼び出されたら、最初に必ず以下のメニューを表示する。
ユーザーが何をすればいいか迷わないようにする。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🚀 LP提案自動生成ツール
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

何ができるか：
  業界名を入力するだけで、競合調査→施策提案→提案書→ワイヤーフレームを自動生成します。

📋 メニュー：
  [1] クイック生成 → 業界名だけ入れて一発で全部作る
  [2] カスタム生成 → 構成を相談しながら作る（おすすめ）
  [3] ブラッシュアップ → 既存の提案書pptxを改善する
  [4] 途中から再開 → 特定のPhaseから始める

💡 使い方の例：
  ・「1 クラウド勤怠管理SaaS」→ 一発で全自動
  ・「2」→ 対話しながらカスタム作成
  ・「3」→ pptxファイルをアップロードして改善
  ・「Phase 3から」→ 分析済みデータをもとにpptxだけ生成

どれにしますか？（番号または自由にどうぞ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## スライド枚数テンプレート

### 5枚（エグゼクティブサマリー）
1. 表紙 → 2. 競合＋差別化 → 3. LP構成案 → 4. FVイメージ → 5. 見積もり

### 10枚（標準）
1. 表紙 → 2. 目次 → 3. 市場 → 4. 競合比較 → 5. 差別化
→ 6. ペルソナ → 7. LP構成案 → 8. FVイメージ → 9. 見積もり → 10. クロージング

### 15枚（詳細）
10枚 + 競合個別2枚 + CTA戦略 + デザイン方針 + SEO提案

### 20枚（フル提案）
15枚 + トレンド + A/Bテスト + KPI + 運用提案 + 体制紹介

### 情報密度
| ライト | 1スライド1メッセージ、余白多め |
| 標準 | 2〜3ポイント＋補足 |
| リッチ | テーブル、グラフ、詳細データ込み |

## 品質基準
- 競合分析は最低5社
- ペルソナは必ず3パターン
- 各セクションに戦略意図
- 提案書は「読まなくても構成がわかる」品質
- ワイヤーは単体で提案に使える品質
